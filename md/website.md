# Website Strategy

คนส่วนใหญ่ไม่ได้ค้นหา

"แผ่นหอมสำหรับโถปัสสาวะ"

แต่ค้นหา

* ห้องน้ำมีกลิ่น
* ห้องน้ำเหม็น
* วิธีดับกลิ่นห้องน้ำ
* ทำอย่างไรให้ห้องน้ำหอม
* กลิ่นห้องน้ำร้านอาหาร
* กลิ่นห้องน้ำโรงแรม

ดังนั้นเว็บไซต์ต้องขาย

Problem → Experience → Solution

ก่อน Product
